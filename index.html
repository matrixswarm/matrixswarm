✨ MatrixSwarm

A file-driven agent OS. No containers. No daemons. Just files.

🚀 Features

  • 🔹 Spawns agents from folders (no Docker, no sockets)
  • 🔹 Delegates with recursive tree logic
  • 🔹 Talks through /comm/agent/incoming file drops
  • 🔹 Fully autonomous, fault-tolerant swarm control
